# XG5000 프로그래밍 가이드

## XG5000 설치 경로
- 실행파일: C:\XG5000\XG5000.exe

## XG5000 개요
- XGT PLC 시리즈 전용 프로그래밍/디버깅 소프트웨어
- 지원 PLC: XGK, XGB, XGI, XGR 시리즈
- 프로젝트 파일 확장자: .xgwx (통합형)
- 무료 제공

## 시스템 요구사항
- 펜티엄 이상 / 256MB 이상 메모리
- RS-232C 또는 USB 포트
- Windows XP/Vista/7/8 지원
- HDD 500MB 이상

## 새 프로젝트 만들기
1. [프로젝트] → [새 프로젝트]
2. 프로젝트 이름 입력
3. CPU 시리즈: XGB 선택
4. CPU 종류: XGB-XBCS → XBC-DR30SU 선택
5. 프로그램 이름 지정

## 프로젝트 구성
- 프로젝트 > PLC > 변수/설명
- 파라미터 (기본, I/O)
- 스캔 프로그램 (Program1, Program2...)
- 태스크 (정주기, 초기화, 외부, 내부)

## LD(래더) 프로그래밍 단축키
- F5: a접점 (평상시 열림, NO)
- F6: b접점 (평상시 닫힘, NC)
- F7: 코일 출력
- F8: 응용명령 (타이머, 카운터 등)
- F10: END 입력, 펑션블록
- Ctrl+L: 라인(렁) 삽입
- Ctrl+D: 라인(렁) 삭제
- Shift+F4: 리셋 코일

## PC-PLC 접속
- USB 케이블(USB-301A) 또는 RS-232C(PMC-310S)
- USB 드라이버: XG5000 설치 시 자동 설치
- 장치관리자에서 "LGIS XGSeries" 확인

## 주요 기능
### 모니터링
- LD 프로그램 모니터
- 변수 모니터
- 시스템 모니터
- 디바이스 모니터
- 트렌드 모니터
- 데이터 트레이스
- PID 모니터

### 디버깅
- 브레이크포인트 설정/해제
- 런(Run), 커서 위치까지 실행
- 스텝 진행
- 디바이스 브레이크
- 스캔 브레이크

### 시뮬레이션 (XG-SIM)
1. I/O 파라미터 설정
2. 시뮬레이터 시작
3. 시스템 모니터로 확인

### 런 중 수정
- 운전 중 프로그램 수정 가능 (RUN 중 수정)

### 강제 I/O
- 입출력 강제 On/Off 기능

## 프로그래밍 언어
- LD (래더 다이어그램) - 주력
- IL (인스트럭션 리스트)
- SFC (순차 기능 차트)
- ST (구조화 텍스트)

## 가상 시뮬레이션 사용 (화성폴리텍 매뉴얼)
1. I/O 파라미터 설정 (1/3)
2. 시뮬레이터 시작 (2/3)
3. 시스템 모니터 (3/3)

## GX Works2 단축키 호환 설정
- XG5000에서 미쓰비시 GX Works2의 단축키와 유사하게 설정 가능
- a접점/b접점/코일 단축키 일치시키기
- Ctrl + 방향키 사용 방법
