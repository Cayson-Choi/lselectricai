# XBC PLC 사양 및 디바이스 영역 정리

## XBC-DR30SU 사양
- CPU: XGB 표준형(SU 타입)
- 입력: 18점 DC24V (소스/싱크)
- 출력: 12점 릴레이
- 전원: AC 100V~240V
- 연산속도: 94ns/step
- 프로그램 용량: 15K step
- 최대 I/O: 284점 (증설 포함)
- 최대 증설: 7단 (옵션 2단 포함, 통신 최대 2단)
- USB 포트 내장
- 크기: 135 x 90 x 64mm

## XBC-DR20SU 사양
- 입력: 12점 DC24V, 출력: 8점 릴레이
- 나머지 사양은 DR30SU와 동일

## 모델명 해석 (XBC-DR30SU)
- XBC: XGB 콤팩트 타입
- D: DC 입력
- R: 릴레이 출력 (N=싱크TR, P=소스TR)
- 30: 총 I/O 30점
- S: 표준형
- U: USB 포트 내장

## 디바이스 메모리 영역 (SU 타입)

### 비트 디바이스
| 디바이스 | 영역 | 점수 | 용도 |
|----------|------|------|------|
| P | P0000~P1023f | 16384점 | 입출력 접점 이미지 |
| M | M0000~M1023f | 16384점 | 내부 보조 릴레이 |
| L | L0000~L2047f | 32768점 | 통신 접점 (고속링크/P2P) |
| K | K00000~K4095f | 65536점 | 정전유지 접점 |
| F | F0000~F1023f | 16384점 | 특수 릴레이 (시스템 플래그) |
| T | T0000~T1023 | 1024점 | 타이머 접점 |
| C | C0000~C1023 | 1024점 | 카운터 접점 |
| S | S00.00~S127.99 | 스텝 컨트롤러 |

### 워드 디바이스
| 디바이스 | 영역 | 크기 | 용도 |
|----------|------|------|------|
| D | D0000~D10239 | 10240워드 | 데이터 레지스터 |
| U | U00.00~U0A.31 | 특수모듈 데이터 |
| Z | Z000~Z127 | 128워드 | 인덱스 레지스터 |
| R | R0000~R10239 | 10240워드 | 파일 레지스터 |

### 래치(정전유지) 가능 디바이스
- 래치 가능: M, T, C, S, D, R
- 기본 래치: K, L, R
- 래치 불가: P, F, U, Z

## 프로그램 구성
- 초기화 프로그램: INIT_DONE까지 실행
- 스캔 프로그램: 반복 실행
- 정주기 인터럽트: 최대 8개
- 외부 인터럽트: SU타입 8개(P000~P007)
- 내부 디바이스 인터럽트: 최대 8개
- 서브루틴: CALL 명령으로 호출

## 스캔 워치독 타이머
- 설정 범위: 10~1000ms (1ms 단위)
- WDT 명령으로 리셋 가능
