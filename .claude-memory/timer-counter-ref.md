# 타이머 & 카운터 레퍼런스

## 타이머 종류 (5종)

### 1. TON (On Delay Timer)
- 입력 On → 현재값 증가 → 설정값 도달 시 출력 On
- 입력 Off → 현재값 0으로 리셋
- XG5000 입력: F10 → TON t0 30

### 2. TOFF (Off Delay Timer)
- 입력 On → 출력 즉시 On, 현재값=설정값
- 입력 Off → 현재값 감소 → 0 도달 시 출력 Off

### 3. TMR (적산 타이머)
- 입력 On 동안만 현재값 증가 (누적)
- 설정값 도달 → 출력 On (RST 명령으로만 Off)
- Shift+F4 → t0 입력 → 리셋 코일 생성

### 4. TMON (Monostable Timer)
- 입력 On(펄스) → 출력 On → 설정시간 후 출력 Off
- 진행 중 재입력 무시

### 5. TRTG (Retriggerable Timer)
- TMON과 유사하나 진행 중 재입력 시 타이머 리셋(처음부터 재시작)

## 타이머 시간 범위
| 종류 | 범위 |
|------|------|
| 100ms 타이머 | 0.1초 ~ 6553.5초 |
| 10ms 타이머 | 0.01초 ~ 655.35초 |
| 1ms 타이머 | 0.001초 ~ 65.535초 |

## 타이머 오차
- 최대: 1 스캔타임 + (스캔시작~타이머명령 실행까지 시간)

## 카운터 종류 (4종)

### 1. CTU (가산 카운터)
- 입력 상승에지에서 현재값 증가
- 현재값 = 설정값 → 출력 On
- 리셋 시 현재값=0, 출력 Off

### 2. CTD (감산 카운터)
- 입력 상승에지에서 현재값 감소
- 현재값 = 0 → 출력 On
- 리셋 시 현재값=설정값, 출력 Off

### 3. CTUD (가감산 카운터)
- 가산입력 에지 → 증가, 감산입력 에지 → 감소
- 현재값 >= 설정값 → 출력 On
- F5(a접점)으로 f99 입력 → 상시연결
- P0: 카운트 업, P1: 카운트 다운

### 4. CTR (링 카운터)
- 현재값 증가 → 설정값 도달 → 다음 에지에서 현재값 0으로 순환
- 현재값 = 설정값일 때 출력 On

## 카운터 최대 계수 속도
- 입력 On/Off 시간 > 스캔타임이어야 카운트 가능
- 고속 펄스는 내장 고속카운터 기능 사용

## XG5000 단축키 (화성폴리텍 매뉴얼 기준)
- F5: 평상시 열린접점 (a접점)
- F6: 평상시 닫힌접점 (b접점)
- F7: 코일
- F8: 응용명령
- F10: END, 펑션블록 입력
- Ctrl+L: 라인 삽입
- Ctrl+D: 라인 삭제
- Shift+F4: 리셋 코일 생성 (t0 입력)
