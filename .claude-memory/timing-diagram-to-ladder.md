# 타이밍 다이어그램 → 래더 변환 패턴 (실습과제 11~20 분석)

## 공통 구조

### 자기유지 회로 (모든 문제 공통)
```
렁0: P0000(PB1) --[a접점]-- P0001(PB2) --[b접점]-- → M0000
     M0000 --[a접점]-- (병렬)                          (자기유지)
```
- PB1: 시작 버튼 (P0000)
- PB2: 정지 버튼 (P0001, b접점으로 차단)
- M0000: 동작 유지용 보조릴레이

### 타이머 설정값 계산 (100ms 기준)
- 1초=10, 2초=20, 3초=30, 4초=40, 5초=50, 9초=90, 10초=100, 15초=150

---

## 패턴 1: 순차 점등 (시간 경과 후 하나씩 ON)
**문제 11, 12 패턴**

하나의 TON으로 최대 시간 설정 → `>=` 비교로 각 출력 시점 제어

### 과제 11: PB1 → RL즉시, 5초후GL, 10초후BZ
```
M0000 → TON T000 100  (총 10초)
M0000 → P0040(RL)     (즉시 ON)
>= T000 50 → P0041(GL) (5초 이상)
>= T000 100 → P0042(BZ) (10초 이상)
```

### 과제 12: PB1 → 2초후RL, +3초후GL, +4초후BZ
```
M0000 → TON T000 90  (총 9초=2+3+4)
>= T000 20 → P0040(RL)  (2초 이상)
>= T000 50 → P0041(GL)  (5초=2+3 이상)
>= T000 90 → P0042(BZ)  (9초=2+3+4 이상)
```
**핵심: 누적 시간으로 비교값 계산**

---

## 패턴 2: 순차 소등 (처음 모두 ON → 하나씩 OFF)
**문제 13 패턴**

하나의 TON → `<=` 비교로 각 소등 시점 제어

### 과제 13: PB1 → RL,GL,BZ 모두ON. 5초후RL off, 10초후GL off, 15초후BZ off
```
M0000 → TON T000 150  (총 15초)
<= T000 50 → P0040(RL)   (5초 이하면 ON)
<= T000 100 → P0041(GL)  (10초 이하면 ON)
<  T000 150 → P0042(BZ)  (15초 미만이면 ON)
```
**핵심: `<=` 사용 → 설정값 이하일 때 ON, 초과하면 OFF**

---

## 패턴 3: 대칭 점등-소등 (ON구간이 중간에 위치)
**문제 14, 15 패턴**

`>=` AND `<=` 범위 비교로 ON 구간 지정

### 과제 14: RL on→3초→GL on→2초→BZ on→5초→BZ off→2초→GL off→3초→RL off
```
M0000 → TON T000 150 (15초=3+2+5+2+3)
T000 --[b접점]-- → P0040(RL)          (타이머 미완료=15초간 ON)
>= T000 30 AND <= T000 120 → P0041(GL)  (3초~12초 구간)
>= T000 50 AND <= T000 100 → P0042(BZ)  (5초~10초 구간)
```

### 과제 15: RL 5초ON→GL 5초ON→BZ ON (순차 교체)
```
M0000 → TON T000 100  (10초)
<= T000 50 → P0040(RL)                  (0~5초)
>= T000 50 AND <= T000 100 → P0041(GL)  (5~10초)
>= T000 100 → P0042(BZ)                 (10초~)
```
**핵심: AND로 두 비교 조합 → ON/OFF 구간 정의**

---

## 패턴 4: 재점등 (OFF 후 다시 ON)
**문제 16 패턴**

OR 조건으로 여러 구간에서 같은 출력 ON

### 과제 16: RL on→5초→RL off+GL on→4초→GL off+RL 다시on
```
M0000 → TON T000 90  (9초=5+4)
(<= T000 50) OR (>= T000 90) → P0040(RL)  (0~5초 OR 9초 이후)
>= T000 50 AND <= T000 90 → P0041(GL)     (5~9초)
```
**핵심: 출력이 다시 켜지면 OR 병렬 조건**

---

## 패턴 5: 독립 2타이머 (시작/정지 별도 시퀀스)
**문제 17 패턴**

PB1용 타이머(T000)와 PB2용 타이머(T001)을 각각 운용

### 과제 17: PB1→RL,3초GL,2초YL / PB2→YL off,2초GL off,3초RL off
```
P0000 AND T001(b접점) → M0000 (자기유지) + TON T000 50
P0001 AND T001(b접점) → M0001 (자기유지) + TON T001 50

M0000 AND T001(b접점) → P0040(RL)
T000 >= 30 [조건] → P0041(GL)
T000 >= 50 → P0042(YL)
```
**핵심: 켜는 시퀀스와 끄는 시퀀스를 별도 타이머로 관리**

---

## 패턴 6: 반복 동작 (사이클 타이머)
**문제 18, 19, 20 패턴 - 가장 중요!**

### 핵심 기법: TON 자동 리셋 반복
```
M0000 --[a접점]-- T000 --[b접점]-- → TON T000 [주기값]
```
- T000이 설정값 도달 → T000 접점 ON → b접점 OFF → 타이머 입력 끊김
- → T000 리셋(0) → T000 접점 OFF → b접점 ON → 타이머 다시 시작
- **결과: 설정값 주기로 무한 반복!**

### 과제 18: RL 2초→GL 3초→BZ 4초 반복 (주기 9초)
```
P0001 --[N에지]-- → M0001 (PB2 하강에지로 정지)
P0000 AND M0001 → M0000 (자기유지)
M0000 AND T000(b접점) → TON T000 90  (9초 사이클 반복)
<= T000 20 → P0040(RL)                  (0~2초)
>= T000 20 AND <= T000 50 → P0041(GL)   (2~5초)
>= T000 50 → P0042(BZ)                  (5~9초)
```

### 과제 19: RL(4초OFF-2초ON), GL(1초후 3초ON-3초OFF), BZ(4초후 3초ON-3초OFF)
```
M0000 (PB1/PB2 자기유지)
M0000 → TON T000 10 (1초 딜레이)

T001(b접점) → TON T001 60  (6초 사이클 반복)
>= T001 40 → P0040(RL)  (4~6초=2초 ON, 0~4초=4초 OFF)

T000(a접점) AND T002(b접점) → TON T002 60  (1초 딜레이 후 6초 사이클)
<= T002 30 → P0041(GL)  (1초후 0~3초 ON)
>= T002 30 → P0042(BZ)  (1초+3초=4초후 ON, 3초 ON)
```
**핵심: 딜레이 타이머 + 사이클 타이머 조합**

### 과제 20: X3 ON, GL(2초OFF-3초ON), BZ(3초ON-2초OFF) 반복
```
M0000 (PB1/PB2 자기유지)
M0000 AND T000(b접점) → TON T000 50  (5초 사이클)
M0000 → P0040(X3)         (항상 ON)
>= T000 20 → P0041(GL)    (2~5초=3초 ON, 0~2초=2초 OFF)
<= T000 30 → P0042(BZ)    (0~3초=3초 ON, 3~5초=2초 OFF)
```
**핵심: 하나의 사이클 타이머로 상호 보완적 패턴 제어**

## 변환 법칙 요약

### 1. 출력 ON 시점 판단
| 타이밍 | 비교 명령 | 예시 |
|--------|-----------|------|
| N초 후 ON (유지) | >= T000 N*10 | 5초 후 ON: >= T000 50 |
| N초까지 ON (이후 OFF) | <= T000 N*10 | 5초까지 ON: <= T000 50 |
| A~B초 구간만 ON | >= AND <= | 3~7초: >= 30 AND <= 70 |
| 즉시 ON (타이머 종료시 OFF) | T000 b접점 | 전체 시간 ON |

### 2. 반복 동작
- **사이클 = TON + 자기 b접점** → 자동 리셋 반복
- 주기 = ON시간 + OFF시간

### 3. 딜레이 시작
- 별도 TON → a접점으로 메인 타이머 트리거

### 4. 교번 동작 (서로 반대)
- 같은 타이머, >= / <= 반대 조건 사용
