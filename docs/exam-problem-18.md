# 전기기능장 PLC 79회 2과제 모의고사 18 분석

## 문제 개요
- 출처: 뱅가의전기놀이터, 모의고사 18
- 난이도: ★★★★★ (배수계산 + BSUM + SR 순차점등)
- 총 스텝: 226개

## 문제 설명
- PBA→N(백의자리), PBB→M(십의자리), PBC→K(일의자리)
- NMK(10진) 구성 → 큰수/중간수/작은수로 SORT
- NMK의 중간수 배수 갯수 + 큰수 배수 갯수 합산
- 합을 2진수 변환 → 1(SET)의 갯수로 램프 점등
- 1의 갯수 > 5: PLA~PLE 1초간 순차 N회 점등 후 정지

### 예시
- N=1,M=4,K=7 → 147, 7의배수=21, 4의배수=36, 합=57=111001(2), 1갯수=4 → PLB~PLE
- N=1,M=3,K=8 → 138, 8의배수=17, 3의배수=46, 합=63=111111(2), 1갯수=6(>5) → 순차점등

### 갯수별 램프 매핑
| 갯수 | 값 | 램프 |
|------|-----|------|
| 1 | 16 | PLE |
| 2 | 24 | PLD+PLE |
| 3 | 28 | PLC+PLD+PLE |
| 4 | 30 | PLB~PLE |
| 5 | 31 | PLA~PLE |

## I/O: P00000~02=PBA~PBC, P00003~05=SSA~SSC, P00040~44=PLA~PLE

## 핵심 알고리즘
1. **BMOV**로 N,M,K를 BCD 자릿수 조합 → **BIN** 변환
2. **FIWRP**로 배열 구축 (D00010에 N,M,K 삽입), **SORT**로 정렬
3. **DIV** 2회: NMK÷큰수, NMK÷중간수 → 배수 갯수
4. **ADD**: 두 배수 갯수 합산
5. **BSUM**: 합의 2진수 1의 갯수 카운트
6. 갯수 ≤ 5: MOVP로 즉시 점등
7. 갯수 > 5: **SR**(시프트 레지스터) + TON 1초로 순차 N회 점등 (D00300 회차카운터)

## 새로 배운 명령어
- **BSUM S D**: 비트합 - S의 2진수에서 1의 갯수 → D
- **ADD S1 S2 D**: 덧셈 S1+S2→D
- **FIWRP S D**: FIFO Write Pulse (배열에 데이터 삽입, D=포인터+데이터)
- **SR S1 S2 S3 N**: Shift Register (S2=입력비트, S3=방향비트, N=비트수)

## 주요 내부 디바이스
- D00300: 순차점등 회차 카운터 (N회 반복 추적)
- D00100: BMOV BCD 조합용
- D00200: BIN 변환 결과 (NMK 10진값)
- D06241.3~.5: N/K/M 입력금지 비트
