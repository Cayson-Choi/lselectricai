# 전기기능장 PLC 79회 2과제 모의고사 34 분석

## 문제 개요
- 출처: 뱅가의전기놀이터, 모의고사 34
- 파일: `전기기능장예제/전기기능장_PLC_2과제_모의고사_34_답.pdf`, `_답_IL.pdf` (6페이지)
- 난이도: ★★★★ (핑퐁 왕복 + 만남 카운터)

## 문제 설명
- N: 시작 위치 (1~5), 입력 제한 N < 6
- M: 동시 점등 횟수 (카운터, 제한 없음)
- N 위치에서 양쪽으로 점등, 1초씩 이동
- PLA/PLE 경계에서 턴(반전), 서로 반대 방향으로 왕복
- 두 포인터가 동시에 1초간 켜진 횟수를 카운터하여 M번 후 정지
- PLA=1, PLB=2, PLC=3, PLD=4, PLE=5
- 1칸 = 1초, 재동작 가능

## I/O 할당
| 디바이스 | 이름 | 기능 |
|----------|------|------|
| P00000 | PBA | N값 입력 (P-edge) |
| P00001 | PBB | M값 입력 (P-edge) |
| P00002 | PBC | 실행 (P-edge) |
| P00003 | SSA | b접점 |
| P00004 | SSB | 시스템 시작 |
| P00005 | SSC | 정지/리셋 (P-edge) |
| P00040~44 | PLA~PLE | 램프 1~5 |

## 핵심 알고리즘: 핑퐁(왕복) + 만남 카운터

### 포인터 경로
```
Z001: N → N+1 → ... → 5 → [턴1] → 4 → ... → 1 → [턴1 해제] → 2 → ... → 5 → ...
Z002: N → N-1 → ... → 1 → [턴2] → 2 → ... → 5 → [턴2 해제] → 4 → ... → 1 → ...
```

### 턴 제어 로직
```
Z001=5 → SET 턴1 (방향 전환: 증가→감소)
Z001=1 AND 턴1 → RST 턴1 (방향 복귀: 감소→증가)
Z002=1 → SET 턴2 (방향 전환: 감소→증가)
Z002=5 AND 턴2 → RST 턴2 (방향 복귀: 증가→감소)
```

### 3연속 인덱스 윈도우 ★★★
```
RST M00000[Z001]     ; Z001 위치 OFF
SET M00001[Z001]     ; Z001+1 위치 ON (현재)
RST M00002[Z001]     ; Z001+2 위치 OFF
```
→ 1칸만 ON인 이동 윈도우 효과
→ P00040 = M00002 OR M00012 (오프셋 +2 주의!)

### 만남 카운터
```
MOVP -1 D00100              ; 초기값 -1 (P-edge, 동작 시작 시 1회)
AND= Z001 Z002 AND NOT T0001 → INCP D00100  ; ★ T0001 b접점으로 중복 카운트 방지
M == D00100 → TON T0010 10 → 리셋    ; 1초 대기 후 리셋
```

## 실행 예시 (N=2, M=2)

| 시간 | Z001 | Z002 | 턴1 | 턴2 | PLA | PLB | PLC | PLD | PLE | D00100 |
|------|------|------|-----|-----|-----|-----|-----|-----|-----|--------|
| 0s | 2 | 2 | - | - | - | ON+ON | - | - | - | -1→0 |
| 1s | 3 | 1 | - | - | ON | - | ON | - | - | 0 |
| 2s | 4 | 2 | - | 턴2 | - | ON | - | ON | - | 0 |
| 3s | 5 | 3 | 턴1 | - | - | - | ON | - | ON | 0 |
| 4s | 4 | 4 | - | - | - | - | - | ON+ON | - | 0→1 |
| 5s | 3 | 5 | - | - | - | - | ON | - | ON | 1 |
| ... | ... | ... | ... | ... | ... | ... | ... | ... | ... | ... |
| M번 만남 → 1초 대기 → 리셋 |

## 추가 사항
- **동작조건**: ANDG> N 0 2 → N>0 AND M>0 동시 확인
- **리셋 시 FMOVP 0 Z000 100**: Z000~Z099 클리어 (다른 문제의 10개보다 넓은 범위)

## 새로 배운 패턴
- **MOVP -1 D**: 음수값 저장 (카운터 사전 초기화)
- **AND= Z001 Z002 AND NOT T0001**: 만남 감지 + 중복 방지
- **3연속 RST/SET/RST**: 인덱스 윈도우 패턴
- **턴(Turn) 플래그**: SET/RST 조합으로 왕복 방향 전환
