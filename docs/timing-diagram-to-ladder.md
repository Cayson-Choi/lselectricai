# 타이밍 다이어그램 → 래더 변환 패턴 (실습과제 11~20 이미지 답안 검증 완료)

## 공통 I/O 매핑
- P0000 = PB1 (시작), P0001 = PB2 (정지)
- M0000 = 동작 유지용 보조릴레이
- P0040 = 첫번째 출력(RL), P0041 = 두번째(GL), P0042 = 세번째(BZ/YL)

## 공통 구조: 자기유지 회로

```
렁0: P0000(a접점) ─── P0001(b접점) ──────────── M0000(코일)
     M0000(a접점) ─┘  (병렬 자기유지)
```

### 타이머 설정값 (100ms 기준)
1초=10, 2초=20, 3초=30, 4초=40, 5초=50, 9초=90, 10초=100, 15초=150

---

## 패턴 1: 순차 점등 (N초 후 하나씩 ON) ★
**과제 11, 12** — 가장 기본 패턴

하나의 TON으로 최대 시간 설정 → `>=` 비교로 각 출력 시점 제어

### 과제 11: PB1 → RL즉시, 5초후GL, 10초후BZ
```
렁: M0000 ─────────────────────── TON T000 100 (10초)
렁: M0000 ─────────────────────── P0040(RL)     ← 즉시 ON
렁: ├ >= T000 50 ├──────────────── P0041(GL)     ← 5초 이상
렁: ├ >= T000 100 ├─────────────── P0042(BZ)     ← 10초 이상
    END
```
✅ 이미지 검증 완료

### 과제 12: PB1 → 2초후RL, +3초후GL, +4초후BZ
```
렁: M0000 ─────────────────────── TON T000 90  (누적 9초=2+3+4)
렁: ├ >= T000 20 ├──────────────── P0040(RL)    ← 2초 이상
렁: ├ >= T000 50 ├──────────────── P0041(GL)    ← 5초(2+3) 이상
렁: ├ >= T000 90 ├──────────────── P0042(BZ)    ← 9초(2+3+4) 이상
    END
```
✅ 이미지 검증 완료
**핵심: 각 시점의 누적 시간으로 비교값 계산**

---

## 패턴 2: 순차 소등 (처음 모두 ON → 하나씩 OFF) ★
**과제 13**

하나의 TON → `<=` 또는 `<` 비교로 각 소등 시점 제어

### 과제 13: PB1 → RL,GL,BZ 모두ON. 5초후RL off, 10초후GL off, 15초후BZ off
```
렁: M0000 ─────────────────────── TON T000 150 (총 15초)
렁: ├ <= T000 50 ├──────────────── P0040(RL)    ← 5초 이하일때 ON
렁: ├ <= T000 100 ├─────────────── P0041(GL)    ← 10초 이하일때 ON
렁: ├ <  T000 150 ├─────────────── P0042(BZ)    ← 15초 미만일때 ON
    END
```
✅ 이미지 검증 완료 (BZ는 `<` 사용, RL/GL은 `<=` 사용)
**핵심: `<=` 사용 → 설정값 이하일 때 ON, 초과하면 OFF**

## 패턴 3: 대칭 점등-소등 (ON 구간이 중간에 위치) ★★
**과제 14, 15** — `>=` AND `<=` 범위 비교

### 과제 14: RL on→3초→GL on→2초→BZ on→5초→BZ off→2초→GL off→3초→RL off
타이밍: 0─────3─────5──────────10────12─────15초
```
렁: M0000 ─────────────────────── TON T000 150 (15초=3+2+5+2+3)
렁: T000(b접점/NC) ────────────── P0040(RL)    ← 타이머 미완료 동안 ON (0~15초)
렁: ├>=T000 30├ ─ ├<=T000 120├ ── P0041(GL)    ← 3초~12초 구간
렁: ├>=T000 50├ ─ ├<=T000 100├ ── P0042(BZ)    ← 5초~10초 구간
    END
```
✅ 이미지 검증 완료
- RL은 T000 b접점(NC) 사용: 타이머 완료 전까지 ON → 15초간 계속 ON

### 과제 15: RL 5초ON → GL 5초ON → BZ ON (순차 교체)
```
렁: M0000 ─────────────────────── TON T000 100 (10초)
렁: ├ <= T000 50 ├──────────────── P0040(RL)                ← 0~5초
렁: ├>=T000 50├ ─ ├<=T000 100├ ── P0041(GL)                ← 5~10초
렁: ├ >= T000 100 ├─────────────── P0042(BZ)                ← 10초~
    END
```
✅ 이미지 검증 완료
**핵심: AND로 두 비교 조합 → ON/OFF 구간 정의**

## 패턴 4: 재점등 (OFF 후 다시 ON) ★★
**과제 16** — OR 조건으로 여러 구간에서 같은 출력 ON

### 과제 16: RL on→5초→RL off+GL on→4초→GL off+RL 다시on
```
렁: M0000 ─────────────────────── TON T000 90 (9초=5+4)
렁: ├<=T000 50├ ──┬────────────── P0040(RL)    ← 0~5초 OR 9초 이후
    ├>=T000 90├ ──┘ (OR 병렬)
렁: ├>=T000 50├ ─ ├<=T000 90├ ── P0041(GL)    ← 5~9초
    END
```
✅ 이미지 검증 완료
**핵심: 출력이 다시 켜지면 OR(병렬) 비교 조건 사용**

## 패턴 5: 독립 2타이머 (시작/정지 별도 시퀀스) ★★★
**과제 17** — 가장 복잡한 패턴

PB1 ON 시퀀스: RL on → 3초 → GL on → 2초 → YL on
PB2 OFF 시퀀스: YL off → 2초 → GL off → 3초 → RL off

### 과제 17 래더 구조 (이미지 검증)
```
렁0: P0000(a) ─── T001(b) ──────── M0000(코일)   ← PB1으로 시작
     M0000(a) ─┘  (자기유지)
렁:  M0000 ─────────────────────── TON T000 50    ← ON 시퀀스 타이머(5초)

렁6: P0001(a) ─── T001(b) ──────── M0001(코일)   ← PB2로 정지 시퀀스 시작
     M0001(a) ─┘  (자기유지)
렁:  M0001 ─────────────────────── TON T001 50    ← OFF 시퀀스 타이머(5초)

렁12: M0000(a) ─ T001(b) ────────── P0040(RL)     ← PB1~T001완료 동안 ON
렁:  ├>=T000 30├ ─ ├<=T001 20├ ─── P0041(GL)     ← PB1+3초~PB2+2초
렁:  ├ >= T000 50 ├────────────── P0042(YL)       ← PB1+5초~ (T001완료시 M0000 해제로 OFF)
     END
```
✅ 이미지 검증 완료

**동작 분석:**
- T001(b접점)이 M0000 자기유지 회로에 삽입 → T001 완료(5초) 시 M0000 강제 해제
- RL: M0000 ON이고 T001 미완료일 때 ON → PB2+5초 후 OFF
- GL: T000>=30(PB1+3초) AND T001<=20(PB2+2초 이내) → 교차 타이머 비교
- YL: T000>=50(PB1+5초) → M0000 해제 시 자동 OFF

**핵심: 켜는 시퀀스(T000)와 끄는 시퀀스(T001) 별도 타이머, 교차 비교**

## 패턴 6: 반복 동작 (사이클 타이머) ★★★ 가장 중요!
**과제 18, 19, 20**

### 핵심 기법: TON 자동 리셋 반복
```
M0000(a접점) ─── T000(b접점/NC) ──── TON T000 [주기값]
```
**원리:**
1. M0000 ON → T000(b접점) 닫힘 → 타이머 시작
2. T000이 설정값 도달 → T000 접점 ON → b접점 열림 → 타이머 입력 OFF
3. T000 리셋(0) → T000 접점 OFF → b접점 닫힘 → 타이머 다시 시작
4. **→ 설정값 주기로 무한 반복!**

---

### 과제 18: RL 2초→GL 3초→BZ 4초 반복 (주기 9초)
**특이사항**: PB2 "눌렀다 놓으면" 정지 → **음변환검출(LOADN)** 사용
- 답안 이미지 접점 기호: `|N|` (Negative/falling edge)
- IL 명령: `LOADN P0001` (LOADP 아님! 실전에서 틀려서 수정한 경험 있음)

```
렁0: P0001 ──[|N|음변환]────────── M0001(코일)    ← PB2 하강에지(놓을 때) 1스캔 펄스
렁3: P0000(a) ── M0001(b) ──────── M0000(코일)    ← PB1 시작, M0001 펄스로 해제
     M0000(a) ─┘
렁:  M0000(a) ── T000(b) ────────── TON T000 90   ← 9초 사이클 자동 반복
렁:  ├ <= T000 20 ├──────────────── P0040(RL)     ← 0~2초 ON
렁:  ├>=T000 20├ ─ ├<=T000 50├ ── P0041(GL)     ← 2~5초 ON (3초간)
렁:  ├ >= T000 50 ├──────────────── P0042(BZ)     ← 5~9초 ON (4초간)
     END
```
✅ 이미지 검증 완료 + XG5000 실전 검증 완료
**핵심: 음변환검출(LOADN)로 PB2 놓는 순간 감지 → M0001 1스캔 펄스로 M0000 해제**

---

### 과제 19: RL(4초OFF-2초ON 반복), GL(1초후 3초ON-3초OFF 반복), BZ(4초후 3초ON-3초OFF 반복)
**특이사항**: 딜레이 타이머 + 복수 사이클 타이머

```
렁0: P0000(a) ── P0001(b) ──────── M0000(코일)    ← 자기유지
     M0000(a) ─┘
렁:  M0000 ─────────────────────── TON T000 10    ← 1초 초기 딜레이

렁:  [M0000연결] ── T001(b) ─────── TON T001 60   ← 6초 사이클 (RL용, 즉시시작)
렁:  ├ >= T001 40 ├──────────────── P0040(RL)     ← 4~6초 ON = 2초ON, 4초OFF

렁14: T000(a) ── T002(b) ──────── TON T002 60    ← 6초 사이클 (GL/BZ용, 1초 딜레이)
렁:  ├ <= T002 30 ├──────────────── P0041(GL)     ← 0~3초 ON (1초후 시작)
렁:  ├ >= T002 30 ├──────────────── P0042(BZ)     ← 3~6초 ON (4초후 시작)
     END
```
✅ 이미지 검증 완료

**동작 타이밍 (검증):**
- T001: 즉시 시작, 6초 반복. RL ON @ 4~6초 → 4초OFF, 2초ON ✓
- T000: 1초 딜레이 타이머
- T002: T000 완료(1초) 후 시작, 6초 반복
  - GL ON @ 0~3초 of T002 → 첫 ON = t=1s~t=4s ✓
  - BZ ON @ 3~6초 of T002 → 첫 ON = t=4s~t=7s ✓
  - GL과 BZ는 같은 타이머의 전반/후반으로 상호 보완!

**핵심: 딜레이 타이머(T000) → 메인 사이클 타이머(T002) 트리거**

---

### 과제 20: X3 항상ON, GL(2초OFF-3초ON 반복), BZ(3초ON-2초OFF 반복)
```
렁0: P0000(a) ── P0001(b) ──────── M0000(코일)    ← 자기유지
     M0000(a) ─┘
렁:  M0000(a) ── T000(b) ────────── TON T000 50   ← 5초 사이클 자동 반복
렁:  ─────────────────────────────── P0040(X3)     ← M0000에서 직접 = 항상 ON
렁:  ├ >= T000 20 ├──────────────── P0041(GL)     ← 2~5초 ON = 3초ON, 2초OFF
렁:  ├ <= T000 30 ├──────────────── P0042(BZ)     ← 0~3초 ON = 3초ON, 2초OFF
     END
```
✅ 이미지 검증 완료

**동작 분석:**
- GL: T000 >= 20 → 2초부터 5초까지 ON (3초), 0~2초 OFF (2초) ✓
- BZ: T000 <= 30 → 0초부터 3초까지 ON (3초), 3~5초 OFF (2초) ✓
- GL과 BZ는 ON 시간 같지만 위상이 다름 (교번 패턴)

**핵심: 하나의 사이클 타이머로 상호 보완적 패턴 제어**

## 변환 법칙 총정리 ★★★

### 1. 출력 ON 시점 판단
| 타이밍 패턴 | 비교 명령 | 예시 |
|-------------|-----------|------|
| N초 후 ON (유지) | `>= T000 N*10` | 5초 후 ON: `>= T000 50` |
| N초까지 ON (이후 OFF) | `<= T000 N*10` | 5초까지 ON: `<= T000 50` |
| A~B초 구간만 ON | `>= AND <=` | 3~7초: `>= 30 AND <= 70` |
| 타이머 전체 시간 ON | T000 b접점(NC) | 타이머 완료 전까지 ON |

### 2. 반복 동작 (자동 리셋)
```
M0000(a) ── T000(b/NC) ── TON T000 [주기값]
```
- 주기 = ON시간 + OFF시간
- 주기 내 위치로 출력 제어 (`>=`, `<=`)

### 3. 딜레이 시작
- 별도 TON(딜레이) → a접점으로 메인 사이클 타이머 트리거
- 과제 19: T000(1초 딜레이) → T002(6초 사이클)

### 4. 교번/보완 동작
- 같은 타이머에서 `>=` / `<=` 반대 조건 → 상호 보완 출력
- 과제 20: GL(`>=20`) vs BZ(`<=30`) → 같은 5초 사이클 다른 위상

### 5. 에지검출 (LOADP/LOADN) 사용
- "눌렀다 놓으면" = **음변환검출(LOADN, 하강에지)** 사용 ← 과제 18 답안 기준
- 1스캔 펄스로 자기유지 해제
- 주의: 문제 텍스트의 "양변환" 표현에 속지 말고, **답안 이미지의 |P| 또는 |N| 기호 확인!**
- `LOADP` = 상승에지 (OFF→ON), `LOADN` = 하강에지 (ON→OFF)

### 6. 2타이머 교차 비교
- 시작 시퀀스와 정지 시퀀스가 독립적일 때 (과제 17)
- 출력 조건에서 두 타이머를 AND로 교차 비교

### 7. 비교명령 없이 a접점/b접점만으로 풀기 ★★★ (XG5000 검증완료)
비교명령(>=, <=) 대신 **타이머를 시점마다 각각 배치**하는 방법.

**변환 규칙:**
| 비교명령 | a접점/b접점 대체 |
|----------|-----------------|
| `>= T000 30` (3초 이상 ON) | 별도 `TON T_x 30` → T_x **a접점** |
| `<= T000 50` (5초 이하 ON) | 별도 `TON T_x 50` → T_x **b접점** |
| `>= A AND <= B` (구간) | `TON T_a A` a접점 AND `TON T_b B` b접점 |

**출력 공식**: `출력 = (ON시점 타이머 a접점) AND (OFF시점 타이머 b접점)`

**장점**: 비교명령 모름/사용 불가 시에도 동일 동작 구현 가능
**단점**: 타이머 수가 증가 (비교명령 1개 = 타이머 1개 추가)

**과제 17 응용 예**: 타이머 2개+비교3개 → 타이머 4개+비교0개 (검증완료)
- 상세 코드: [IL→래더 변환 패턴](il-to-ladder-patterns.md) → "비교명령 없이" 섹션
