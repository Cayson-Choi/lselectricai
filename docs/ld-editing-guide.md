## 5.2 프로그램 편집

### 5.2.1 편집 도구 - 단축키 총정리 ★★★

#### 접점 단축키
| 단축키 | 기능 |
|--------|------|
| **F3** | 평상시 열린 접점 (NO, a접점) |
| **F4** | 평상시 닫힌 접점 (NC, b접점) |
| **Shift+F1** | 양 변환 검출 접점 (상승엣지) |
| **Shift+F2** | 음 변환 검출 접점 (하강엣지) |

#### OR 접점 단축키
| 단축키 | 기능 |
|--------|------|
| **Ctrl+3** | 평상시 열린 OR 접점 |
| **Ctrl+4** | 평상시 닫힌 OR 접점 |
| **Ctrl+5** | 양 변환 검출 OR 접점 |
| **Ctrl+6** | 음 변환 검출 OR 접점 |

#### 선(Line) 단축키
| 단축키 | 기능 |
|--------|------|
| **F5** | 가로선 (수평선) |
| **F6** | 세로선 (수직선) |
| **Shift+F8** | 연결선 |
| **Shift+F9** | 반전 입력 |

#### 코일 단축키
| 단축키 | 기능 |
|--------|------|
| **F9** | 코일 (일반 출력) |
| **F11** | 역 코일 |
| **Shift+F3** | 셋(Latch) 코일 |
| **Shift+F4** | 리셋(Unlatch) 코일 |
| **Shift+F5** | 양 변환 검출 코일 |
| **Shift+F6** | 음 변환 검출 코일 |

#### 펑션/명령어 단축키
| 단축키 | 기능 |
|--------|------|
| **F10** | 펑션/펑션 블록 |
| **Shift+F7** | 확장 펑션 |

#### 커서 이동 (재정의 불가)
| 단축키 | 기능 |
|--------|------|
| Home | 열의 시작으로 |
| End | 열의 끝으로 |
| Ctrl+Home | 프로그램 시작으로 |
| Ctrl+End | 편집된 마지막 줄로 |
| Back space | 현재 데이터 삭제 + 왼쪽 이동 |
| 방향키 | 상하좌우 한 칸 이동 |

#### 편집 공통 단축키
| 단축키 | 기능 |
|--------|------|
| **Esc** | 선택 모드로 변경 |
| **Ctrl+Z** | 편집 취소 (횟수 제한 없음) |
| **Ctrl+Y** | 재 실행 |

> **참고**: 단축키 표기에서 s = Shift, c = Ctrl (예: sF1 = Shift+F1)
> 사용자 정의 단축키 설정: 제2장 2.4절 참고

### 5.2.2 접점 입력
1. 접점 입력할 위치로 커서 이동
2. 도구모음에서 선택 또는 단축키 (F3/F4/Shift+F1/Shift+F2)
3. 변수 입력 대화상자에서 디바이스명 입력 → 확인

**팁:**
- 즉시 입력 모드 해제 시 변수 입력 대화상자 표시 안됨 ([도구]-[옵션]-[LD/IL 편집])
- Enter 키: 이전에 입력한 것과 같은 종류의 접점/코일 재입력

### 5.2.3 OR 접점 입력
- OR 연결할 위치로 커서 이동 → Ctrl+3/4/5/6
- **주의**: 겹침 모드에서 커서 위치에 접점이 있으면 OR 연결 안됨

### 5.2.4 변수/디바이스 입력
**대화상자 구성:**
- 변수/디바이스: 디바이스 또는 변수명 입력
- 변수/설명 목록: 선언된 변수/설명 표시
- 플래그: 시스템/고속링크/P2P/PID 플래그 선택
- 변수/설명 자동 추가: 미등록 디바이스 입력 시 자동 추가 여부
- 자동완성: 입력 문자로 시작하는 변수/디바이스 자동 완성

**디바이스 자동할당 프로젝트:**
- 변수 선택 대화상자가 별도 표시
- 상수, 직접 변수, 선언 변수 입력 가능
- 로컬 변수 목록에서 추가/편집/삭제 가능

### 5.2.5 선 입력
- **수평선(F5)**: 요소 간 가로 연결
- **수직선(F6)**: 요소 간 세로 연결 (현재 커서 기준 **왼쪽 아래 방향**으로 입력)

### 5.2.6 코일 입력
- 위치 이동 → F9/F11/Shift+F3~F6 → 디바이스명 입력
- **자동 수평선**: 코일/출력 응용 명령어 입력 시 왼쪽 요소와 자동 연결
- **이중 코일 주의**: 같은 프로그램 내 이중 코일 시 마지막 코일 값으로 출력

### 5.2.7 응용 명령어 입력 (F10)
1. 위치 이동 → F10 또는 [편집]-[응용 명령어]
2. 응용 명령어 대화상자:
   - 응용명령: 명령어 입력
   - 분류: 카테고리별 필터링 (전체 선택 시 모든 명령어)
   - 명령어 리스트: 분류별 명령어 표시
   - 사용방법: 오퍼랜드별 가능 영역 표시
   - 명령어 도움말: 선택 명령어 설명
3. 오퍼랜드 입력 팁:
   - 명령어 입력 후 Enter → 변수/디바이스 대화상자 자동 표시
   - 마지막 오퍼랜드가 아니면 커서 자동 이동

### 5.2.8 펑션/펑션 블록 입력 (Shift+F7)
- 디바이스 자동할당 프로젝트에서만 사용 가능
- 확장 펑션 대화상자: 이름 검색, 분류별 리스트, 입력 개수 설정

### 5.2.9 설명문 입력
**렁 설명문:**
- [편집]-[설명문/레이블 입력] → 설명문 선택 → 내용 입력 → 확인
- 렁의 시작 위치에 표시

**출력 설명문:**
- 출력 요소 위치에서 더블 클릭 또는 Enter
- 출력 요소가 있는 경우에만 입력 가능

### 5.2.10 레이블 입력
- [편집]-[설명문/레이블 입력] → 레이블 선택
- JMP 명령어에서 참조
- **제한**: 영문 16자 / 한글 8자
- 대소문자 구별, 첫 글자는 숫자/특수문자 불가

### 5.2.11~5.2.15 삽입/삭제
| 작업 | 메뉴 | 비고 |
|------|------|------|
| 셀 삽입 | [편집]-[셀 삽입] | 렁 단위 동작 |
| 라인 삽입 | [편집]-[라인 삽입] | 영역 선택 시 선택 라인 수만큼 삽입 |
| 요소 삭제 | [편집]-[삭제] | 접점/코일/명령어/선/설명문/레이블 |
| 셀 삭제 | [편집]-[셀 삭제] | 렁 단위, OR 요소 있으면 불가 |
| 라인 삭제 | [편집]-[라인 삭제] | 선택 영역 전체 삭제 |

### 5.2.16 복사/잘라내기/붙여넣기
- 복사: 영역 선택 → [편집]-[복사] → 위치 이동 → [편집]-[붙여넣기]
- 잘라내기: 영역 선택 → [편집]-[잘라내기] (원본 삭제)

### 5.2.17 드래그&드롭
- **이동**: 영역 선택 → 마우스 왼쪽 누른 채 드래그 → 위치에서 해제
- **복사**: Ctrl 키 + 드래그 (원본 유지)
- **변수/설명 붙여넣기**: 변수/설명 창에서 드래그 → 접점/코일/명령어 오퍼랜드에 드롭
  - 오퍼랜드 타입이 다르면 붙여넣기 불가
  - 변수 모니터/트렌드 모니터 창으로도 복사 가능

### 5.2.18 편집 취소 / 재실행
- Ctrl+Z: 편집 취소 (**횟수 제한 없음**)
- Ctrl+Y: 재 실행

### 5.2.19 접점 수 조절
- [보기]-[접점 수 변경]: 9/12/16/20/24/28/32 중 선택
- 보기 툴바에서 증가/감소 가능
- 범위: **9 ~ 32**
- 접점 수보다 큰 렁은 '▶ 연결번호' 표시로 다음 라인 연결

### 5.2.20 LD 화면 속성
- [보기]-[LD 화면 속성]
- 디바이스/변수/설명문 보기 옵션, 배율, 접점 수 일괄 설정

## 5.3 프로그램 보기

### 5.3.1 IL 프로그램으로 보기
- [보기]-[IL] 선택 → LD를 IL로 변환 표시/편집
- **주의**: 미완성 렁이 있으면 IL 변환 불가

### 5.3.2 배율 변경
- 확대: [보기]-[화면 확대] 또는 Ctrl+아래쪽 휠
- 축소: [보기]-[화면 축소] 또는 Ctrl+위쪽 휠

### 5.3.3~5.3.7 오퍼랜드 보기 모드
| 메뉴 | 표시 내용 |
|------|-----------|
| [보기]-[디바이스 보기] | 디바이스명만 |
| [보기]-[변수 보기] | 변수 선언 시 변수명, 아니면 디바이스명 |
| [보기]-[디바이스/변수 보기] | 디바이스 + 변수 모두 |
| [보기]-[디바이스/설명문 보기] | 디바이스 + 설명문 |
| [보기]-[변수/설명문 보기] | 변수 + 설명문 |

## 5.4 편집 부가 기능

### 5.4.1 프로그램 최적화
- [편집]-[프로그램 최적화]
- 접점 간 불필요한 가로선 및 빈 라인 삭제
- **주의**: 실행 취소 불가! LD 로직 미완성 시 동작 안함

### 5.4.2 비 실행문
- 설정: [편집]-[비 실행문 설정]
- 해제: [편집]-[비 실행문 해제]
- 비 실행문 영역은 프로그램 용량에 미포함, 설명문 용량에 포함
- PLC 쓰기/읽기 가능 (설명문과 동일)
- **런 중 수정 불가** → [쓰기] 메뉴로 설명문 쓰기 필요

### 5.4.3 북 마크
- 설정/해제: [편집]-[북 마크]-[설정/해제] (토글)
- 모두 해제: [편집]-[북 마크]-[모두 해제]
- 이전/다음 이동: [편집]-[북 마크]-[이전/다음 북마크]
- **라인 단위** 설정 (동일 프로그램 내에서만 이동)
- 편집 취소/재실행에 포함 안됨

### 5.4.4 찾아가기
| 기능 | 메뉴 경로 |
|------|-----------|
| 스텝 찾아가기 | [찾기/바꾸기]-[찾아가기]-[스텝] |
| 레이블 찾아가기 | [찾기/바꾸기]-[찾아가기]-[레이블] |
| 렁 설명문 찾아가기 | [찾기/바꾸기]-[찾아가기]-[렁 설명문] |
| END 명령어 찾아가기 | [찾기/바꾸기]-[찾아가기]-[END 명령어] |

- 프로그램 전체 선택 시 모든 항목 검색 가능

## LD 편집 핵심 요약 (시험 대비) ★★★

### 가장 많이 쓰는 단축키
1. **F3** → a접점(NO)
2. **F4** → b접점(NC)
3. **F9** → 코일
4. **F10** → 펑션/펑션블록 (타이머, 카운터 등)
5. **F5** → 가로선
6. **F6** → 세로선
7. **Shift+F3** → 셋(SET) 코일
8. **Shift+F4** → 리셋(RST) 코일

### 주의사항
- 이중 코일: 마지막 코일 값으로 출력
- 코일/출력 명령어 입력 시 수평선 자동 생성
- 수직선은 왼쪽 아래 방향
- 프로그램 최적화는 실행 취소 불가
- 미완성 렁은 IL 변환 불가
