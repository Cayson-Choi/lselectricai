# 전기기능장 PLC 79회 2과제 모의고사 31 분석

## 문제 개요
- 출처: 뱅가의전기놀이터, 모의고사 31
- 파일: `전기기능장예제/전기기능장_PLC_2과제_모의고사_31_답.pdf`, `_답_IL.pdf` (8페이지)
- 난이도: ★★★★★ (MUL/SUB 시간 계산 + 이중 포인터 ON/OFF 분리)

## 문제 설명
- N(시작위치 1~5), M(소등기준위치 1~5) 입력, 재동작 가능
- 동작 조건: 0 < N,M, 입력제한 N,M < 6
- PLA=1, PLB=2, PLC=3, PLD=4, PLE=5
- 1칸 = 0.5초

### 동작 해석 (N=2, M=4 예시) ★★★ 검증완료
**① "N에서 시작 상하 1초 지연 지속 점등"**
- N=2(PLB)에서 시작, 위아래로 1초마다 1칸씩 확장
- 0초: PLB ON → 1초: PLA+PLC ON → 2초: PLD ON → 3초: PLE ON
- 한번 켜진 램프는 계속 유지 (지속 점등)

**② "PLA, PLE중 마지막 램프 1초 점등 뒤"**
- PLA는 1초에 도달, PLE는 3초에 도달 → 마지막 = PLE
- PLE 켜진 후 5개 전체 ON 상태로 1초 대기

**③ "M에서 먼쪽에서 먼저 1초 지연 소등 시작"**
- M=4(PLD) 기준 거리: PLA(1)까지=3칸(먼쪽), PLE(5)까지=1칸(짧은쪽)
- 먼쪽(PLA 방향) 먼저 소등 시작
- 짧은쪽(PLE 방향)은 늦게 시작 → 두 방향이 M에서 동시 도착

**④ "M에서 최종 소등"**
- OFF+0초: PLA OFF → +1초: PLB OFF → +2초: PLC+PLE OFF → +3초: PLD OFF(최종)

## 리셋 구조 (이중 조건)
```
LOAD 과제2 / ANDP SSC → ; SSC P-edge 리셋
OR NOT 과제2             ; ★ 과제2 OFF 시에도 리셋 (시스템 OFF)
→ FMOVP 0 D00000 10000 / FMOVP 0 Z000 10 / BRST M00000 100
```

## T0010 자동리셋 (재동작용)
```
LOADP T0010 → FMOVP 0 Z000 10 / BRST M00000 100 / FMOVP 0 D00015 10000
```
OFF 구간 완료(T0010 타임업) 후 자동 초기화 → 재동작 가능

## 출력부 (NOT 이중반전 패턴)
```
; 래더: M00001(a접점) → P00040
; IL: LOAD NOT M00001 / NOT / OUT P00040  (이중반전 = a접점)
```

## 핵심 알고리즘: ON/OFF 구간 분리

### 연산 (시간 계산)
```
D00003 = M × 10           ; M값에 0.5초 단위 10을 곱함 (타이머 설정값)
D00005 = 50 - D00003      ; 5초 - M시간 (한쪽 거리)
D00006 = D00003 - 10      ; M시간 - 1초 (다른쪽 거리)
먼위치의시간 = MAX(D00005, D00006)
짧은거리의초 = MIN(D00005, D00006)
짧은거리시작시간 = 먼위치의시간 - 짧은거리의초
모드A: M >= 3 (PLA 방향에서 증가하며 소등 → M에서 최종 소등)
모드B: M < 3 (PLE 방향에서 감소하며 소등 → M에서 최종 소등)
```

예시 (N=2, M=4):
- D00003 = 40, D00005 = 10, D00006 = 30
- 먼위치의시간 = 30, 짧은거리의초 = 10, 짧은거리시작시간 = 20

### ON 구간
1. Z001, Z002 모두 N에서 시작 (FMOVP N Z001 2)
2. Z001 증가(위), Z002 감소(아래) → 양방향 확장 점등
3. Z001=5 또는 Z002=1 도달 시 해당 방향 정지
4. 양쪽 정지 후 1초 대기 → ON 구간 완료
5. ★ SET M00000[Z001/Z002] 조건에 `AND NOT 온정지` 포함 (ON 완료 후 더 이상 점등 안함)

### OFF 구간
1. Z001=6, Z002=0 으로 초기화
2. 모드별로 먼쪽부터 소등 시작
3. 짧은거리시작시간 도달 시 반대쪽도 소등 시작
4. T0010 = 먼위치의시간 도달 → 완료

## 새로 배운 명령어
- **MUL S1 S2 D**: 곱셈 (S1 × S2 → D)
- **SUB S1 S2 D**: 뺄셈 (S1 - S2 → D)
- **BRST S N**: 일괄 리셋 (연속 실행, BRSTP의 비펄스 버전)
- **NOT**: 누산기 반전 (출력 패턴에서 사용)
