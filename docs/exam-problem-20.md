# 전기기능장 PLC 79회 2과제 모의고사 20 분석

## 문제 개요
- 출처: 뱅가의전기놀이터, 모의고사 20
- 난이도: ★★★★★ (배열 처리 + 검색/삭제 + 2진수 출력)
- 총 스텝: 132개

## 문제 설명
- 1~20 수 자동 등록
- PBA(P): N회 입력 → 중복 수이면 PLA 점등유지
- PBB(P): 리스트에서 삭제 (중복수일때 삭제불가)
- PBC(P): K값 입력 (기준값, 1~20)
- 동작조건: 삭제 갯수 ≥ 3, K > 0
- SSC(P): N 입력값 제외, K보다 높은 수의 갯수를 2진수로 PLB~PLE 표시
- SSC(N): 리셋/정지

### 예시
- 제외{7,3,8,2,11}, K=6 → 6보다 높은 수 11개 → 01011(2) → PLB,PLD,PLE
- 제외{...,4,14,9,19}, K=13 → 5개 → 00101(2) → PLC,PLE

## I/O: P00000~02=PBA~PBC, P00003~05=SSA~SSC, P00040~44=PLA~PLE

## 핵심 알고리즘
1. **FIWR**로 1~20 배열에 자동 등록 (D00099 카운터 + AND< 20 반복)
2. PBA: **SCH**로 중복 검색 → 중복이면 PLA 점등 (MOVP 16 D00000)
3. PBB: **SCH**로 검색 + **FIDEL**로 배열에서 삭제
4. SSC(P): **DETECT**로 K보다 큰 값의 갯수 검출
5. MOVP D00101→D00000: 갯수를 출력 레지스터로 전송
6. D00000.0~.4 → P00040~44 (2진수 직접 출력)

## 새로 배운 명령어
- **DETECT S1 S2 D N**: 배열 검출 - S1부터 N개에서 S2보다 큰 값의 갯수/위치
- **SDR S1 S2 D N**: Search Data Register - 배열에서 값 검색
- **FIDEL S1 S2 D**: File Delete - 배열에서 요소 삭제
- **FIMR S D**: File Move Register - 배열에 데이터 등록

## 특징
- 타이머 미사용! (배열 처리형 문제)
- 배열 조작 명령어(DETECT/SDR/FIDEL/FIMR) 집중 사용
