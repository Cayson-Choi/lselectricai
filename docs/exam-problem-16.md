# 전기기능장 PLC 79회 2과제 모의고사 16 분석

## 문제 개요
- 출처: 뱅가의전기놀이터, 모의고사 16
- 파일: `전기기능장예제/전기기능장_PLC_2과제_모의고사_16_답.pdf`(4p), `_답_IL.pdf`(8p)
- 난이도: ★★★★★ (승곱 계산 + BCD 자릿수 추출 + 5램프 숫자 표현)
- 총 스텝: 322개, 19개 렁

## 문제 설명
- **3개 입력**: PBA(P)→N, PBB(P)→M, PBC(P)→K
- 동작조건: 0 < N, M, K
- SSC(P): 동작 시작 (계산 + 표시)
- SSC(N): 리셋 및 동작정지
- PBA(N): 동작 중 재동작
- 동작중 N,M,K 입력불가
- 결과값 > 9999 → 자동 리셋, 재동작 가능
- 1칸 = 0.5초

### 승곱 규칙
| 조건 | 계산 | 예시 |
|------|------|------|
| 3개 동일 | 동일수³ | N=4,M=4,K=4 → 4³=64 |
| N이 2개 | N² | N=5,M=2,K=5 → 5²=25 |
| M이 2개 | M² | N=3,M=5,K=5 → 5²=25 |
| 모두 다름 | MAX^MIN | N=9,M=4,K=3 → 9³=729 |
| 결과>9999 | 자동리셋 | N=8,M=6,K=5 → 8⁵=32768 → 리셋 |

### 5램프 숫자 표현 (D00000 비트패턴)
```
비트:  4    3    2    1    0
      PLA  PLB  PLC  PLD  PLE

숫자  패턴      값   램프
 0   11011     27   PLA+PLB+PLD+PLE (사잇값)
 1   00001      1   PLE
 2   00011      3   PLE+PLD
 3   00111      7   PLE+PLD+PLC
 4   01111     15   PLE+PLD+PLC+PLB
 5   11111     31   PLE+PLD+PLC+PLB+PLA (전체)
 6   11000     24   PLA+PLB
 7   11100     28   PLA+PLB+PLC
 8   11110     30   PLA+PLB+PLC+PLD
 9   10101     21   PLA+PLC+PLE (P패턴)
```
- 1~5: 아래(PLE)부터 위로 누적 점등
- 6~8: 위(PLA)부터 아래로 누적 점등
- 9(P): 교차 패턴
- 0: 가운데 빠진 패턴

## 핵심 알고리즘

### 1단계: 중복 검출 (SCH 2회)
```
SCH N N D00030 3        ; [N,M,K]에서 N값 검색 → D00031=일치수
IF D00031=3 → 삼승곱    ; 3개 모두 같음
IF D00031=2 → 이승곱    ; N이 2개 (AND NOT 이승곱2)
IF D00031=1:
  SCH M N D00035 3      ; [N,M,K]에서 M값 검색 → D00036=일치수
  IF D00036=2 → 이승곱2 ; M이 2개 (AND NOT 이승곱)
  IF D00036=1 → 각자수  ; 모두 다름
```

### 2단계: 정렬 + 타입변환
```
SORT D00011 ... → D00011=MAX, D00013=MIN
I2L N D00100     ; N → DWORD
I2L M D00110     ; M → DWORD
I2L D00011 D00120 ; MAX → DWORD
I2L D00013 D00130 ; MIN → DWORD
```

### 3단계: 승곱 계산 (EXPT)
```
삼승곱: EXPT D00100 3 D00050      ; N³
이승곱: EXPT D00100 2 D00050      ; N²
이승곱2: EXPT D00110 2 D00050     ; M²
각자수: EXPT D00120 D00130 D00050 ; MAX^MIN
```

### 4단계: 결과 처리
```
L2I D00050 D00055     ; DWORD → WORD (결과값)
BCD D00055 D00060     ; BIN → BCD 변환
BMOV D00060 D00070 hC004  ; 천의자리 추출
BMOV D00060 D00071 H8004  ; 백의자리 추출
BMOV D00060 D00072 H4004  ; 십의자리 추출
BMOV D00060 D00073 H0004  ; 일의자리 추출
```

### 5단계: 자릿수 표시
```
Z001 인덱스로 D00069[Z001] (=D00070~73) 순회
- digit > 0: SET 중간일때, 시간설정=15 (1.5초)
- digit = 0 AND NOT 중간일때: 시간설정=0 (선행 0 건너뜀)
- digit = 0 AND 중간일때: MOVP 27 (사잇값 0 표시)
- T0001 > 10: MOVP 0 D00000 (자릿수 사이 0.5초 소등)
- 구간설정=4 AND D00000=0: 표시 완료 → 정지 (소등 상태에서만)
```

### 자릿수 표시 타이밍 (1.5초/자릿수)
```
0~1.0초: 해당 숫자 패턴 표시
1.0~1.5초: 전체 소등 (자릿수 전환 간격)
→ 총 1.5초 × 유효자릿수 = 전체 표시 시간
```

## 새로 배운 명령어
- **SCH S1 S2 D N**: 검색 - S2부터 N개에서 S1값 검색, D=첫위치, D+1=일치수
- **EXPT S1 S2 D**: 승곱 - S1^S2 → D (DWORD)
- **I2L S D**: 정수→롱 변환 (WORD→DWORD)
- **L2I S D**: 롱→정수 변환 (DWORD→WORD)
- **BCD S D**: BIN→BCD 변환
- **BMOV S D param**: 비트무브 - 특정 비트 추출
- **SORT S1 S2 S3 S4 S5**: 정렬 (배열 정렬)
- **FIWRP S D**: FIFO Write Pulse (배열에 데이터 삽입)
- **ANDN S**: AND 음변환 (N-edge, falling edge)

## 리셋 조건 (3가지)
1. SSC(N): 음변환 리셋 (ANDN SSC)
2. NOT 과제2: 시스템 OFF
3. D00055 > 9999: 결과값 초과 자동 리셋

## 과제 31~34와의 차이점
- 입력 3개 (N,M,K) vs 2개 (N,M)
- 포인터 이동이 아닌 **산술 계산** (EXPT, SORT, SCH)
- BCD 변환 + 비트 추출로 **자릿수별 표시**
- PBA(N) 재동작 (음변환으로 동작 트리거)
- 자릿수 인덱싱 (D00069[Z001])으로 순차 표시
