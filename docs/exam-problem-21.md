# 전기기능장 PLC 79회 2과제 모의고사 21 분석

## 문제 개요
- 출처: 뱅가의전기놀이터, 모의고사 21
- 난이도: ★★★★ (MIN/MUL + 비트마스크 + 구간제어)
- 총 스텝: 210개

## 문제 설명
- PBA(P)→N회(1~5), PBB(P)→M회(1~5), PBC(P)→A=MIN(N,M) 계산
- 1칸 = 1초
- PLE부터 A번째 램프까지 0.5초 ON/OFF A회 반복 → 지속점등
- A회 반복 후 상위 램프도 동일 패턴
- A번째 램프 완료 → 모든 램프 소등
- 재동작 가능

### 비트패턴→램프 매핑
| 값 | 2진수 | 램프 |
|----|-------|------|
| 16 | 10000 | PLE |
| 8 | 01000 | PLD |
| 4 | 00100 | PLC |
| 2 | 00010 | PLB |
| 1 | 00001 | PLA |

## I/O: P00000~02=PBA~PBC, P00003~05=SSA~SSC, P00040~44=PLA~PLE

## 핵심 알고리즘
1. **MIN** N D00010 2: A=MIN(N,M)
2. **MUL** A 20 D00020: 시간입력=A×20 (A×2초, 100ms 단위)
3. T0001(시간입력): 구간 메인타이머, T0002(20=2초): 0.5초 ON/OFF 깜빡임
4. T0002=10(1초) 시점: MOVP 0→D00000 (반주기 소등)
5. 구간설정(D06242) 증가로 PLE→PLD→PLC→PLB→PLA 순차 처리
6. 소등 마스크: D00100에 이전 구간 램프값 MOVP → 지속점등 표현

## 출력부 (De Morgan OR)
```
LOAD NOT D00000.x   ; 깜빡임 비트
AND NOT D00100.x    ; 지속점등 마스크
NOT                 ; = D00000.x OR D00100.x
OUT P0004x
```

## 핵심 IL 패턴
- **AND= + MOVP 조합**: A값별 분기 처리 (MV= 단일 명령어가 아닌, AND= 비교 + MOVP 전송 조합)
  - 예: `AND= D00010 1 / MOVP 16 D00000` (A=1이면 PLE 점등)
- **De Morgan OR 출력**: LOAD NOT + AND NOT + NOT = OR
- **MIN S D N**: 최소값 검출 (N,M 중 작은 값 → A)

## 특징
- 2개 타이머 조합 (메인 + 깜빡임)
- D00000(깜빡임) + D00100(지속점등) 비트마스크 기법
- De Morgan OR 출력: NOT(NOT A AND NOT B) = A OR B
