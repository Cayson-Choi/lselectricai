# 전기기능장 PLC 79회 2과제 모의고사 19 분석

## 문제 개요
- 출처: 뱅가의전기놀이터, 모의고사 19
- 난이도: ★★★★★ (BCD 덧셈/뺄셈 + 인덱스 룩업 + 다단계 입력)
- 총 스텝: 342개 (가장 긴 문제!)

## 문제 설명
- PBA→N회(십의자리), PBB→M회(일의자리) → K입력 후 → PBA→A회, PBB→B회
- PBC(P): K 입력 (1=덧셈, 2=뺄셈, 입력제한 1~2)
- PBC(N): A,B 중 하나 입력 후 동작 시작
- NM(10) ± AB(10) 계산 → 결과 자릿수별 5램프 표시
- 결과 > 99: 타임차트대로 동작 후 정지/자동리셋
- K=1(덧셈): PLA 동작전 지속점등, K=2(뺄셈): PLE 동작전 지속점등
- 뺄셈 결과 < 0: 동작과 동시 자동리셋

### 5램프 숫자 표현 (모의16과 동일!)
```
0→27, 1→1, 2→3, 3→7, 4→15, 5→31, 6→24, 7→28, 8→30, 9→21
```

## I/O: P00000~02=PBA~PBC, P00003~05=SSA~SSC, P00040~44=PLA~PLE

## 내부 디바이스 (13개 비트 플래그!)
| 비트 | 기능 |
|------|------|
| D06241.0 | 과제2 |

| D06241.2 | 동작조건 |

## 핵심 알고리즘
1. 다단계 입력: N,M → K → A,B → PBC(N)으로 동작
2. **BMOV**로 N,M을 BCD 십/일 자리 조합 → D00010
3. **BMOV**로 A,B를 BCD 십/일 자리 조합 → D00011
4. K=1: **ADDB** D00010 D00011 D00015 (BCD 덧셈)
5. K=2: **SUBB** D00010 D00011 D00015 (BCD 뺄셈)
6. **BIN** D00015 D00030 (BCD→BIN 변환, 99 비교용)
7. **BMOV**로 결과 십/일 자리 분리 → D00021, D00022
8. Z011 인덱스로 D00020[Z011] 참조 → 자릿수별 패턴 MOVP
9. T0001(2초 사이클 = 1초ON + 1초OFF), Z011 순환 (1→2→1→2)
10. >99: T0002(1초)로 구간1~4 순차 표시 후 리셋1

## 새로 배운 명령어
- **ADDB S1 S2 D**: BCD 덧셈 (S1+S2→D, BCD 형식)
- **SUBB S1 S2 D**: BCD 뺄셈 (S1-S2→D, BCD 형식)
- **ANDN S**: AND 음변환 (PBC 놓을 때 동작 트리거)
