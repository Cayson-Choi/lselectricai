# XG5000 제6장 - IL(Instruction List) 편집 가이드

## 6.1 IL 편집기 화면 구성

### IL 창 구성 요소
- **렁(Rung)**: 각 라인이 몇 번째 렁인지 표시 (편집할 때마다 변환 가능)
- **스텝**: 각 라인의 스텝 번호 표시
- **명령어**: 명령어 표시
- **오퍼랜드**: 보기 모드에 따라 디바이스/변수/플래그 표시
- **브레이크 포인터**: 디버그용 중단점 표시
- **렁 설명문**: 렁에 대한 주석/설명
- **비 실행문**: 실행하지 않는 라인 표시
- **에러 렁 표시**: 미완성 렁 표시 (완성 시 사라짐)
- **북 마크**: 책갈피 위치 표시
- **선택 라인**: 현재 선택된 라인 강조

### 명령 입력 창
- 명령어와 오퍼랜드를 입력/수정하는 전용 창
- 제한사항: 출력 설명문은 표시/편집 불가

### 색상 설정
- [도구] → [옵션]에서 선택 라인 색, 배경색, 비 실행문 색, 에러 렁 표시 색 변경 가능

## 6.2 프로그램 편집

### 6.2.1 겹침 모드 / 삽입 모드
| 모드 | 동작 | 설명 |
|------|------|------|
| **겹침 모드** | 기존 내용 수정(덮어쓰기) | [편집] → [겹침 모드] |
| **삽입 모드** | 새 내용 추가, 기존 내용은 다음 라인으로 이동 | [편집] → [삽입 모드] |

- XG5000 좌측 하단에 현재 모드 표시
- 마지막 라인 편집 시 항상 삽입 모드로 동작
- 겹침/삽입 모드는 명령어 입력(6.2.2)과 응용 명령어 입력(6.2.3)에서만 사용
- 렁 설명문 입력(6.2.4)과 붙여넣기(6.2.8)는 항상 삽입 모드

### 6.2.2 명령어 입력
1. 입력할 라인 선택
2. Enter → 명령 입력 창 생성
3. 명령어와 오퍼랜드 입력
4. Enter → 입력 완료, 다음 라인에 명령 입력 창 자동 생성

**핵심 규칙:**
- 명령어와 오퍼랜드 사이: **SPACE 문자**로 구분
- 오퍼랜드와 오퍼랜드 사이: **SPACE 문자**로 구분
- 자동완성: 명령어 입력 시 해당 문자로 시작하는 명령어 리스트 자동 생성 → Enter로 선택
- 완성된 렁만 저장됨 (에러가 있는 렁은 저장 안됨)
- 에러 렁이 있으면 LD로 전환 불가

### 6.2.3 응용 명령어 입력
두 가지 방법:
1. 직접 입력 (6.2.2 방식)
2. [편집] → [응용 명령어] 선택 → 대화상자에서 입력

**문자열 상수 입력:**
- 작은 따옴표 사용
- 예: `$MOV '가나다' m1`

### 6.2.4 렁 설명문 입력
- [편집] → [설명문 입력] → 내용 입력 → 확인
- 렁 설명문은 항상 **렁의 맨 처음**에 위치
- 렁 중간에 입력해도 자동으로 렁 첫 라인으로 이동

### 6.2.5 명령어 수정
1. 수정할 라인 선택
2. Enter → 명령 입력 창에 기존 명령어/오퍼랜드 표시
3. 수정
4. Enter → 완료

### 6.2.6 라인 삽입
- 삽입할 위치 라인 선택 → [편집] → [라인 삽입]
- 지정한 라인 수만큼 삽입 가능
- 단축키: Ctrl+L (LD와 동일)

### 6.2.7 라인 삭제
- 삭제할 라인 선택 → [편집] → [라인 삭제]
- 명령어, 렁 설명문, 비 실행문 관계없이 삭제
- 맨 마지막 라인에서는 삽입/삭제 불가
- 단축키: Ctrl+D (LD와 동일)

### 6.2.8 붙여넣기, 복사, 잘라내기
- **잘라내기**: 라인 선택 → [편집] → [잘라내기] (선택 라인 삭제됨)
- **복사**: 라인 선택 → [편집] → [복사] (선택 라인 유지)
- **붙여넣기**: 붙여넣을 위치 선택 → [편집] → [붙여넣기]
- 붙여넣기는 항상 삽입 모드로 동작

### 6.2.9 편집 취소 / 재실행
- **취소**: [편집] → [편집 취소] (Ctrl+Z)
- **재실행**: [편집] → [재 실행] (Ctrl+Y)
- 적용 범위: 명령어 입력, 설명문 입력, 삭제, 수정, 잘라내기/붙여넣기, 디바이스 바꾸기, 문자열 바꾸기, 비 실행문 설정/해제

## 6.3 프로그램 보기

### 6.3.1 배율 변경
- 확대: [보기] → [화면 확대] 또는 Ctrl + 마우스 아래 스크롤
- 축소: [보기] → [화면 축소] 또는 Ctrl + 마우스 위 스크롤
- 보기 도구 모음에서 배율 직접 입력 가능

### 6.3.2~6.3.5 오퍼랜드 보기 모드
| 메뉴 | 표시 내용 |
|------|-----------|
| [보기] → [디바이스 보기] | 오퍼랜드를 디바이스로만 표시 |
| [보기] → [변수 보기] | 변수 선언되면 변수, 아니면 디바이스 |
| [보기] → [디바이스/변수 보기] | 디바이스와 변수 모두 표시 |
| [보기] → [디바이스/설명문 보기] | 디바이스와 설명문 함께 표시 |

## 6.4 편집 부가 기능

### 6.4.1 비 실행문
- PLC에서 실행하지 않는 라인 설정 (코드 주석처리와 유사)
- **설정**: 라인 선택 → [편집] → [비 실행문 설정]
- **해제**: 라인 선택 → [편집] → [비 실행문 해제]
- **렁 단위**로 설정/해제 (선택 라인이 포함된 렁 전체 적용)
- 활용: 나중에 다시 사용할 코드를 임시로 비활성화

### 6.4.2 북 마크 (책갈피)
- **설정/해제**: 라인 선택 → [편집] → [북 마크] → [설정/해제] (토글)
- **모두 해제**: [편집] → [북 마크] → [모두 해제]
- **이전 북마크**: [편집] → [북 마크] → [이전 북마크] (현재 라인 앞쪽)
- **다음 북마크**: [편집] → [북 마크] → [다음 북마크] (현재 라인 뒤쪽)

## 6.5 찾아가기

### 6.5.1 스텝 찾아가기
- [찾기/바꾸기] → [찾아가기] → [스텝]
- 프로그램 선택 → 스텝 번호 입력 → 찾아가기

### 6.5.2 레이블 찾아가기
- [찾기/바꾸기] → [찾아가기] → [레이블]
- 프로그램 선택 → 레이블 목록에서 선택 → 찾아가기

### 6.5.3 렁 설명문 찾아가기
- [찾기/바꾸기] → [찾아가기] → [렁 설명문]
- 프로그램 선택 → 설명문 목록에서 선택 → 찾아가기

### 6.5.4 END 명령어 찾아가기
- [찾기/바꾸기] → [찾아가기] → [END 명령어]
- 프로그램 선택 → END 명령어 목록에서 선택 → 찾아가기

## IL vs LD 비교 핵심
- IL은 텍스트 기반 프로그래밍 언어
- LD는 그래픽 기반 래더 다이어그램
- **상호 변환 가능** (단, IL에 에러 렁이 없어야 LD 변환 가능)
- IL에서는 완성된 렁만 저장됨
- 출력 설명문은 IL에서 표시/편집 불가 (LD에서만 가능)
