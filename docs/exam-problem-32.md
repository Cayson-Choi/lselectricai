# 전기기능장 PLC 79회 2과제 모의고사 32 분석

## 문제 개요
- 출처: 뱅가의전기놀이터, 모의고사 32
- 파일: `전기기능장예제/전기기능장_PLC_2과제_모의고사_32_답.pdf`(2p), `_답_IL.pdf`
- 난이도: ★★★★ (BCD 연산 + 비트 매핑)

## 문제 설명
- N(십의 자리), M(일의 자리), K(배수 기준) 입력
- NM = N×10+M 의 2자리 수를 만들고, K의 배수 개수를 계산
- 배수 개수 = floor(NM ÷ K)
- 결과를 5개 램프에 **2진수**로 표시
- 결과 > 31이면 모든 램프 **교대 점멸 무한반복**
- 1칸 = 0.5초

### 램프-비트 대응
| 램프 | 비트 위치 | 가중치 |
|------|-----------|--------|
| PLA | D00000.4 | 2^4 = 16 |
| PLB | D00000.3 | 2^3 = 8 |
| PLC | D00000.2 | 2^2 = 4 |
| PLD | D00000.1 | 2^1 = 2 |
| PLE | D00000.0 | 2^0 = 1 |

### 계산 예시 검증
| N | M | K | NM | NM÷K | 2진수 | 점등 램프 | 모드 |
|---|---|---|-----|------|-------|----------|------|
| 4 | 7 | 3 | 47 | 15 | 01111 | PLB,PLC,PLD,PLE | 이하31 |
| 6 | 6 | 7 | 66 | 9 | 01001 | PLB,PLE | 이하31 |
| 9 | 6 | 5 | 96 | 19 | 10011 | PLA,PLD,PLE | 이하31 |
| 0 | 8 | 2 | 8 | 4 | 00100 | PLC | 이하31 |
| 7 | 4 | 1 | 74 | 74 | >31 | 교대점멸 | 초과31 |

## 프로그램 구조 (답안 래더 2페이지)

### Rung 1: 시스템 시작
```
LOAD NOT SSA(P00003) AND SSB(P00004) → OUT D06241.0
```

### Rung 5: 리셋
```
과제2(D06241.0) AND SSC(P00005, N-edge) → FMOVP 0 D00000 10000
과제2(D06241.0, b접점) → (추가 리셋/시스템 OFF)
```

### Rung 14: 입력부
```
과제2(D06241.0) AND NOT 동작(D06241.1):
  ├─ PBA(P-edge) AND NOT 입력금지N(D06241.6) → INCP D00003(N)
  ├─ PBB(P-edge) AND NOT 입력금지M(D06241.8) → INCP D00002(M)
  ├─ PBC(P-edge) AND NOT 입력금지K(D06241.7) → INCP D00001(K)
  └─ SSC(P-edge) AND 동작조건(D06241.2) → SET 동작(D06241.1)
```

### Rung 40: 입력 제한 + 동작 조건
```
과제2(D06241.0):
  G> D00001 0 2 → SET 동작조건(D06241.2)    ; K>0 AND M>0 (그룹 비교)
  = D00003 9 → SET 입력금지N(D06241.6)       ; N=9이면 입력 차단
  = D00002 9 → SET 입력금지M(D06241.8)       ; M=9이면 입력 차단
  = D00001 9 → SET 입력금지K(D06241.7)       ; K=9이면 입력 차단
```

### Rung 60: 핵심 연산 ★★★
```
과제2(D06241.0) →
  UNI D00002 D00010 2     ; M(일의자리)+N(십의자리) → BCD 합성
  BIN D00010 D00020        ; BCD → 이진수 변환 (NM값)
  DIV D00020 D00001 D00050 ; NM ÷ K → 몫(D00050)
  (<= D00050 31) → SET 이하31(D06241.3)
  (> D00050 31) → SET 초과31(D06241.4)
```

**연산 흐름 (N=4, M=7, K=3):**
1. UNI: D00002(7) + D00003(4) → D00010 = 0x47 (BCD)
2. BIN: 0x47 → D00020 = 47 (이진수)
3. DIV: 47 ÷ 3 → D00050 = 15 (몫)
4. 15 ≤ 31 → 이하31 모드

### Rung 80: 출력 제어

**이하31 모드 (≤31):**
```
동작(D06241.1) AND 이하31(D06241.3) → MOVP D00050 D00000
; 몫을 D00000에 복사 → 비트가 자동으로 램프에 매핑됨
```

**초과31 모드 (>31): 교대 점멸**
```
초과31(D06241.4) AND NOT T0001 → TON T0001 20  ; 2초 자동리셋 타이머

= T0001 0  → MOVP 10 D00000   ; 0.0s: 0b01010 → PLB, PLD ON
= T0001 5  OR = T0001 15 → MOVP 0 D00000  ; ★ IL: LOAD=/OR=/AND LOAD 패턴
= T0001 10 → MOVP 21 D00000   ; 1.0s: 0b10101 → PLA, PLC, PLE ON
; T0001 = 20 → 타이머 리셋 → 반복
```

점멸 패턴:
```
시간:  0.0s   0.5s   1.0s   1.5s   2.0s(반복)
PLA:   OFF    OFF    ON     OFF    OFF
PLB:   ON     OFF    OFF    OFF    ON
PLC:   OFF    OFF    ON     OFF    OFF
PLD:   ON     OFF    OFF    OFF    ON
PLE:   OFF    OFF    ON     OFF    OFF
       ────   ────   ────   ────   ────
D값:    10      0     21      0     10
```

### Rung 114~126: 최종 출력 (D 비트 → 램프)
```
D00000.4 → P00040 (PLA)
D00000.3 → P00041 (PLB)
D00000.2 → P00042 (PLC)
D00000.1 → P00043 (PLD)
D00000.0 → P00044 (PLE)
```

### Rung 129: END

## 새로 배운 명령어/개념

### 1. UNI (Unite, BCD 합성)
```
UNI S D N    ; S부터 N개 단위 BCD 값을 D에 합성
UNI D00002 D00010 2  ; D00002(일의자리) + D00003(십의자리) → D00010(BCD)
```
- 개별 BCD 숫자(0~9)를 하나의 BCD 워드로 합침

### 2. BIN (BCD→Binary 변환)
```
BIN S D      ; BCD값 S를 이진수로 변환하여 D에 저장
BIN D00010 D00020  ; BCD 0x47 → Binary 47
```

### 3. DIV (나눗셈)
```
DIV S1 S2 D  ; S1 ÷ S2 → 몫을 D에 저장
DIV D00020 D00001 D00050  ; NM ÷ K → 몫
```
- 정수 나눗셈 (나머지는 D+1에 저장됨)

### 4. G> (그룹 비교)
```
G> S C N     ; S부터 N개 연속 워드가 모두 C보다 큰지 확인
G> D00001 0 2  ; D00001>0 AND D00002>0
```

### 5. D 레지스터 비트→출력 직접 매핑
```
D00000.4 → P00040  ; D00000의 bit4가 PLA 직접 구동
```
- 계산 결과를 D 레지스터에 넣으면 비트가 자동으로 램프에 매핑
- 별도의 비트 분해 로직 불필요!

### 6. MOVP를 이용한 타이머 기반 패턴 생성
- 타이머 특정 시점에서 MOVP로 D00000 값을 변경
- D 비트가 출력에 매핑되어 있으므로 패턴이 자동 표시

## 모의고사 32 vs 33 비교

| 항목 | 32번 | 33번 |
|------|------|------|
| 난이도 | ★★★★ | ★★★★★ |
| 입력 | N,M,K (3개) | N,M (2개) |
| 핵심 연산 | UNI+BIN+DIV (BCD 합성+나눗셈) | MAX+MIN+인덱스 포인터 |
| 출력 방식 | D 비트→램프 직접 매핑 | 2레이어 M 릴레이 OR |
| 동적 요소 | 이하31: 정적 표시 / 초과31: 점멸 | 이중 포인터 스위프 |
| 프로그램 크기 | ~130 렁 (2페이지) | ~228 렁 (3페이지) |
| 새 명령어 | UNI, BIN, DIV, G> | FMOVP, BRSTP, <=3, JMP/LBL |
| 공통 명령어 | MOVP, INCP, FMOVP, TON | MOVP, INCP, DECP, FMOVP, TON |
