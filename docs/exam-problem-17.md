# 전기기능장 PLC 79회 2과제 모의고사 17 분석

## 문제 개요
- 출처: 뱅가의전기놀이터, 모의고사 17
- 난이도: ★★★★★ (FIFO큐 + 시프트레지스터 + SKIP + 가변타이머)
- 총 스텝: 165개

## 문제 설명
- PLA→PLE 순차 1초 점등 무한 반복
- **PBA(P)**: SKIP - 현재 램프 즉시 다음으로, 이후 반복에서 해당 램프 SKIP
- **PBB(P)**: 입력된 램프부터 점등 시간 1초 증가
- **PBC(P)**: 입력된 다음 램프부터 점등 시간 1초 감소 (최소 1초)
- 최종 램프 1개만 남으면 PBA/PBB/PBC 입력금지
- SSC(P): 동작, SSC(N): 리셋/정지

## I/O 할당
| 디바이스 | 이름 | 기능 |
|----------|------|------|
| P00000 | PBA | SKIP 입력 |
| P00001 | PBB | 시간 증가 |
| P00002 | PBC | 시간 감소 |
| P00003 | SSA | 시작 |
| P00004 | SSB | 시작 |
| P00005 | SSC | 동작/리셋 |
| P00040~44 | PLA~PLE | 램프 |

## 핵심 알고리즘
1. **FIWRP**로 FIFO 큐에 순서값(1,2,4,8,16) 기록 (비트 가중치)
2. **SCHP**로 현재 활성 비트 위치 검색
3. PBA 시 **FIDEL**로 큐에서 해당 순서 삭제 (SKIP)
4. PBB: **ADDP** 입력시간+10, PBC: **SUBP** 입력시간-10 (AND>= 20 가드)
5. **SR**(시프트 레지스터)로 D00100.0~.4 비트 시프트 → 순차 점등
6. 가변 타이머: TON T0001 D06243 (런타임 변경 가능)
7. 간접 주소: **#D00000** (D00000 값을 주소로 사용)

## 새로 배운 명령어
- **FIWRP S D**: FIFO Write Pulse (큐에 데이터 기록)
- **FIDEL D S1 S2**: FIFO Delete (큐에서 삭제)
- **SCHP S1 S2 D1 D2**: Search One-Hot Pulse (활성 비트 검색)
- **SR S1 S2 S3 N**: Shift Register (비트 시프트, N=비트수)
- **ADDP S1 S2 D**: 펄스 덧셈
- **SUBP S1 S2 D**: 펄스 뺄셈
- **#D00000**: 간접 주소지정 (D00000 값을 주소로 참조)

## SKIP 이중 구조
- **D00000 기반 FIFO 큐**: 순서 큐 (1,2,4,8,16) - FIDEL로 삭제
- **D00030 기반 SKIP 리스트**: SKIP할 숫자 별도 FIWRP 기록
- 매 전환 시 SCH로 출력이 SKIP리스트에 있는지 검색 → 있으면 시간설정=0 (즉시 건너뜀)

## SR 방향 제어 (D00101)
- D00101=0: 하강 (좌시프트), D00101=2: 상승 (D00101.1=1, 우시프트)
- 출력이 큐 마지막 도달 → D00101=0, 큐 처음 도달 → D00101=2

## PBB vs PBC 적용 시점
- **PBB**: ADDP 후 즉시 MOVP → 시간설정에 반영 (현재 램프부터)
- **PBC**: SUBP만 (MOVP 없음) → 다음 전환 시 반영 (다음 램프부터)

## 출력부
- D00100.0~.4 → LOAD NOT / NOT → P00040~44 (이중부정 패턴)
