# 전기기능장 PLC 79회 2과제 모의고사 26 분석

## 문제 개요
- 출처: 뱅가의전기놀이터, 모의고사 26
- 파일: `전기기능장예제/전기기능장_PLC_2과제_모의고사_26_답.pdf`(4p), `_답_IL.pdf`(10p)
- 난이도: ★★★★ (구간 카운터 + 조건별 지연/점등시간 분기)
- 총 스텝: 281개 (24렁)
- 특징: 자동리셋 타이머(T0001) + 동적 시간설정(MOVP 분기) + 5개 개별 OFF 타이머

## 문제 설명
- PBA(P) → N 입력 (1~5), PBB(P) → M 입력 (1~5)
- 입력제한: N,M < 6, 동작조건: 0 < N,M
- PBC(P) → 동작 시작
- SSC(P) → 리셋 및 동작정지
- PLA부터 PLE까지 **순차적으로** 점등
- 1칸 = 1초

### 핵심 규칙
- **N,M에 해당하는 램프**: 작은수 초 지연 후 큰수 초간 점등
- **그 외 램프**: 1초 지연 1초간 점등
- PLA=1, PLB=2, PLC=3, PLD=4, PLE=5

### 계산 예시 검증
**N=2, M=4 (작은수=2, 큰수=4):**
| 램프 | 위치 | 유형 | 시작 | 점등시간 |
|------|------|------|------|---------|
| PLA | 1 | 그 외 | 0초 | 1초 |
| PLB | 2=N | N/M | 2초 | 4초 |
| PLC | 3 | 그 외 | 3초 | 1초 |
| PLD | 4=M | N/M | 5초 | 4초 |
| PLE | 5 | 그 외 | 6초 | 1초 |

**N=5, M=2 (작은수=2, 큰수=5):**
| PLA(그 외) 0초/1초 | PLB(M=2) 2초/5초 | PLC(그 외) 3초/1초 | PLD(그 외) 4초/1초 | PLE(N=5) 6초/5초 |

## 핵심 알고리즘

### 1단계: 연산 (Rung 6)
```
MIN N 작은수 2          ; 작은수 = MIN(N,M)
MAX N 큰수 2            ; 큰수 = MAX(N,M)
MUL 작은수 10 지연시간   ; 작은수 × 0.1초 = 작은수 초
MUL 큰수 10 온타임       ; 큰수 × 0.1초 = 큰수 초
SUB 작은수 1 A          ; A = 작은수-1 (AB구간 위치1)
SUB 큰수 1 B            ; B = 큰수-1 (AB구간 위치2)
```

### 2단계: AB구간 + 그 외 판별 (Rung 6 계속)
```
; AB구간: 현재 구간이 N/M 램프 직전인지
LOAD= 구간 A / OR= 구간 B / AND LOAD → OUT AB구간

; "그 외" 플래그: 해당 위치가 N도 M도 아닌 램프
AND<> 1 N / AND<> 1 M → OUT M00011   ; 위치1이 그 외이면 ON
AND<> 2 N / AND<> 2 M → OUT M00012
AND<> 3 N / AND<> 3 M → OUT M00013
AND<> 4 N / AND<> 4 M → OUT M00014
AND<> 5 N / AND<> 5 M → OUT M00015
```

### 3단계: 구간 전환 타이머 (Rung 8-9) ★★★
```
; A=0 (작은수=1): T0100 1초 초기 지연 → PLA 시작 전 1초 대기
AND= A 0 → TON T0100 10

; A>0 (작은수>1): 패스=ON → 즉시 시작 (PLA at t=0)
AND> A 0 → OUT 패스

; 동적 시간설정: AB구간이면 작은수초, 아니면 1초
AND AB구간 → MOVP 지연시간 시간설정     ; 작은수 초
AND NOT AB구간 → MOVP 10 시간설정       ; 1초

; ★ 자동리셋 타이머: T0001 타임업 → 구간++ → T0001 리셋 → 반복
(T0100 OR 패스) AND 동작 AND NOT T0001 → TON T0001 시간설정
                                          INCP 구간
```

**자동리셋 메커니즘:**
1. AND NOT T0001(출력)=TRUE → T0001 시작
2. T0001 카운트 중 → 출력=0 → 조건 유지
3. T0001 타임업 → 출력=1 → AND NOT T0001=FALSE → 입력 OFF → T0001 리셋
4. T0001 출력=0 → AND NOT T0001=TRUE → INCP 구간 (P-트리거) → 다음 구간
5. 새 구간의 시간설정으로 T0001 재시작 → 반복

### 4단계: 램프 ON (Rung 11)
```
AND= 구간 1 → SET M00001   ; PLA ON
AND= 구간 2 → SET M00002   ; PLB ON
AND= 구간 3 → SET M00003   ; PLC ON
AND= 구간 4 → SET M00004   ; PLD ON
AND= 구간 5 → SET M00005 + SET 리셋준비   ; PLE ON + 완료
```

### 5단계: 램프 OFF 타이머 (Rung 13~17, 5개 반복)
```
; 각 램프별 개별 OFF 타이머 (예: PLA)
LOAD 동작 / AND>= 구간 1
  AND M00011 → MOVP 10 시간          ; "그 외" → 1초
  AND NOT M00011 → MOVP 온타임 시간   ; N/M → 큰수 초
  TON T0011 시간
  AND NOT T0011 → OUT M00021          ; 타이머 미만료 → ON 유지
```
★ 구간≥x 조건으로 해당 구간 도달 후 계속 동작
★ M0001x(그 외) 여부에 따라 다른 ON 시간 설정

### 6단계: 출력 (Rung 19~23)
```
LOAD NOT M00001 / NOT / AND M00021 → OUT P00040
; = M00001(ON 플래그) AND M00021(미만료) → PLA
```
이중반전(LOAD NOT + NOT) + AND 미만료 플래그

### 7단계: 자동 리셋 (Rung 11 하단)
```
리셋준비 AND NOT P00040~44 → OUT 리셋
; 모든 램프 OFF 확인 → 리셋 트리거 → FMOVP 전체 초기화
```

## 실행 흐름 (N=2, M=4)
```
작은수=2, 큰수=4, A=1, B=3
지연시간=20(2초), 온타임=40(4초)
패스=ON (A>0 → 즉시 시작)

t=0: 구간=1 → SET M00001(PLA ON), T0011 시작(1초)
     AB구간=(1=1)=ON → 시간설정=20 → T0001 2초 카운트
t=1: T0011 만료 → PLA OFF
t=2: T0001 만료 → 구간=2 → SET M00002(PLB ON), T0012 시작(4초)
     AB구간=OFF → 시간설정=10 → T0001 1초 카운트
t=3: T0001 만료 → 구간=3 → SET M00003(PLC ON), T0013 시작(1초)
     AB구간=(3=3)=ON → 시간설정=20 → T0001 2초 카운트
t=4: T0013 만료 → PLC OFF
t=5: T0001 만료 → 구간=4 → SET M00004(PLD ON), T0014 시작(4초)
     AB구간=OFF → 시간설정=10 → T0001 1초 카운트
t=6: T0001 만료 → 구간=5 → SET M00005(PLE ON), T0015 시작(1초) + 리셋준비
     PLB OFF (T0012 만료, 2~6초=4초간)
t=7: T0015 만료 → PLE OFF
t=9: T0014 만료 → PLD OFF (5~9초=4초간)
t=9+: 리셋준비 AND 모든 램프 OFF → 리셋 → 자동 초기화
```

## 프로그램 구조
| 섹션 | 렁 | 기능 |
|------|------|------|
| 리셋 | 1 | FMOVP (SSC P-edge + OR NOT 과제2 + OR 리셋) |
| 입력 | 3~4 | SSB/SSA→과제2, PBA→N, PBB→M, PBC→동작 |
| 연산 | 6 | MIN/MAX/MUL/SUB + AB구간 + 그 외 판별 |
| 동작 | 8~9 | A=0 분기, 동적 시간설정, 자동리셋 타이머+INCP |
| ON 구간 | 11 | 구간별 SET M0000x + 리셋준비 |
| OFF 구간 | 13~17 | 5개 개별 타이머 (M0001x 분기 → 시간 MOVP) |
| 출력 | 19~23 | M0000x AND M0002x → PLx (이중반전) |
| END | 24 | |

## 새로 배운 패턴

### 1. 자동리셋 타이머 (AND NOT T → TON T)
```
AND NOT T0001 → TON T0001 시간설정 / INCP 구간
```
- T0001 타임업 → 출력 1 → 입력 OFF → 리셋 → 출력 0 → 입력 ON → 재시작
- 구간 카운터를 자동으로 증가시키는 핵심 메커니즘
- ★ 실습과제의 단순 타이머 비교와 완전히 다른 접근!

### 2. MOVP 조건 분기 (동적 타이머 설정)
```
AND AB구간 → MOVP 지연시간 시간설정
AND NOT AB구간 → MOVP 10 시간설정
```
- 같은 디바이스(시간설정)에 조건에 따라 다른 값을 저장
- TON이 매 스캔 시간설정을 읽으므로 동적 변경 가능

### 3. "그 외" 램프 판별 (AND<> x N / AND<> x M)
```
AND<> 1 N / AND<> 1 M → OUT M00011
```
- 위치 x가 N도 M도 아니면 "그 외" 램프
- 5개 위치 × 2개 비교 = 10개 AND<> 명령

### 4. 자동 리셋 완료 감지
```
리셋준비 AND NOT P00040~44 → OUT 리셋
```
- 모든 램프가 꺼진 것을 직접 출력 접점(P0004x)으로 확인
- 리셋 → FMOVP → 재동작 가능 상태

### 5. A=0 특수 처리 (T0100 vs 패스)
```
AND= A 0 → TON T0100 10    ; 작은수=1 → 1초 초기 지연
AND> A 0 → OUT 패스          ; 작은수>1 → 즉시 시작
```
- 작은수=1일 때는 첫 번째 램프가 N/M이므로 지연 필요
- 작은수>1일 때는 첫 번째 램프가 "그 외"이므로 즉시 시작

## 이 문제의 핵심 통찰
1. **구간 카운터 패턴**: INCP + 자동리셋 타이머로 순차 제어 (인덱스 레지스터 미사용)
2. **동적 시간설정**: AB구간 여부에 따라 MOVP로 타이머 설정값 분기
3. **3레이어 출력**: ON플래그(M0000x) AND 미만료(M0002x) → 출력
4. **5개 독립 OFF 타이머**: 각 램프마다 개별 타이머로 서로 다른 ON 시간 구현
5. **자동 완료 감지**: 리셋준비 + 출력 b접점 5개 AND → 리셋 트리거
